## How to Run
1. Clone the repository:  
   ```bash
   git clone https://github.com/JahnaviChakka/Cold-email-automation
   ```
   ```bash
   cd cold-email-generator
   ```

2. Install dependencies:  
   ```bash
   pip install -r requirements.txt
   ```
3. Run the app:  
   ```bash
   streamlit run app.py
   ```
